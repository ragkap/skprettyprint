    Notes

    • URL-encode spaces in tickers: DBS SP → DBS%20SP (or wrap the URL in quotes).
    • 404 if no primer exists for that ticker; 500 on generation failure. Both return JSON {"detail": "…"}.
    • Expect 10–20s response time (DB query + yfinance chart + WeasyPrint render).
    • No auth on endpoints — do not expose publicly without an API key.
